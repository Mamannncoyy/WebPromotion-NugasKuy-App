The Nugas-Kuy! Application

"Nugas-Kuy!" is a groundbreaking application tailored to enhance academic processes, streamlining assignment management, submission, and grading for both students and lecturers. Its online submission and deadline reminder features ensure efficiency and prevent missed assignments, while its comprehensive grading system facilitates transparent feedback. Acting as a communication platform, it fosters interaction between students and lecturers, clarifying expectations and assessment criteria. By optimizing lectures and promoting collaboration, "Nugas-Kuy!" not only revolutionizes assignment handling but also lays the groundwork for digital transformation in higher education.

Benefit

Several Benefits for Users

Organize Materials and Assignments Neatly

Materials and assignments are neatly organized in this application without fear of being lost or deleted because it uses cloud storage.

Online Learning Service

Lecturers no longer need to meet students directly, just give material and assignments in this application and then can immediately assess them.

Learning Schedule is Organized

Students and lecturers have a deadline agreement in submitting assignments in this application so that students can manage which ones are done first.

Deadline Reminder

This application can remind the deadline for submitting assignments, students can set when and how many times the reminder notification sounds so as to minimize forgetting to submit assignments.

Privacy Protected

Assignment assessment results are kept private, including a private chat room that cannot be seen by other students.

How To Use The Nugas-Kuy! App



Login Accounts

Log in with the account that has been given by the campus. For lecturers, the account given uses a username and password. while for students, login using NIM and the default password 123456. Especially students are required to change the default password with their own password without being known to other students in the settings menu

Start Assignments

Lecturers can start assignments at lecture meetings by uploading assignments and materials that have been discussed.

Accepting Assignments and Organizing Them

In the student account, you can see the materials and assignments that have been uploaded by the lecturer. If there are things you want to discuss, you can go to the "students' comments" menu, if you want to set a notification as a deadline reminder, you can set it by pressing the "set deadline notifications" button and if the assignment has been done, please upload it by pressing the "upload assignment answers" button.

Receive Assignments Results and Grade Them

The lecturer's account will show students' answers to their assignments in the "assignment submission results" button that can be assessed directly. then can provide answers to complaints and joint discussions in the "students' comments" button. Furthermore, if the lecturer uploads the wrong assignment, you can press the "edit file or rearrange deadlines" button and can also rearrange the deadline if there are complaints and have been discussed by editing it in that button too.

Ask Anything About Assignments Privately

The chat button can be used to ask questions, request transparency of grades or other things by chatting between students and lecturers privately without other students knowing.

Featured - Student Upload Assignment Answers

The button "upload assignment answers" is a button to upload the answers to the assignments given by the lecturer. There is a check mark to notify that the assignment has been completed.

Asking About Assignment

The button "student comments" is a button to provide comments on the assignment given. Whether it's confused about the assignment given or asking for a waiver of the deadline given.


Set Deadline Notifications

The button "set deadline notifications" is a button to set notifications to remind the assignment deadline, either set a day or two before the deadline or set to provide notifications every second.


Room Chat Private

The chat button on the bottom right is a button to send messages to lecturers privately without other students knowing, either sending messages to request transparency of grades or anything related to lectures

Featured - Lecturer Upload Materials and Assignments

Upload button with a "+" icon in the bottom right corner to upload materials and assignments at lecture meetings.

View Assignment Result and Assess assignment

The button that says "assignment submission results" to see the assignments that students have submitted and then grade them. There is a notification if there are tasks that have not been assessed.


Reply Students' Comments

The button that says "students' comments" to see student comments about the assignment given, whether it is confused about the questions given or asking for relief from the deadline given


Rearrange The Assignment File or Deadline

The button that says "edit file or rearrange deadlines" is a button to rearrange the assignment file if there are errors and to rearrange the deadline schedule given due to several factors including leniency for students who provide comments and have been discussed again


Room Chat Private

he chat button next to the upload button is useful for chatting between lecturers and students privately without other students knowing. It can be to ask for transparency of grades on their assignments or ask other things.
